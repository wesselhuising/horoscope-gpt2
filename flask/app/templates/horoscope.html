Wat is dit en hoe werkt dit

U wilt toch ook weten wat de dag gaat brengen? Gelukkig kan de computer tegenwoordig alles voorspellen en vanaf nu ook uw daghoroscoop door middel van een GPT-2 model. Dit model is getraind op 10.000 unieke Nederlands geschreven horoscopen en is gemaakt om op basis van een beetje input een horoscoop te genereren. Alle uitkomsten zijn daarom niet door mij bedacht maar door het model dat gehost wordt op een server van TransIP. Ik heb een artikel met uitleg en code over de werking van dit model geschreven wat hier te vinden is.

Deze tool demonstreert de werking van het GPT-2 model, beschreven in het artikel, dat is gemaakt voor het genereren van horoscopen in het Nederlands.
